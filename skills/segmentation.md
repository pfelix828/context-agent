# Skill: Segment Analysis

## Description
Breaks down key metrics by customer segment (SMB, Mid-Market, Enterprise) to identify where the business is winning and where there are gaps.

## When to Use
- "How do segments compare?"
- "Which segment has the best win rate?"
- "Where should sales focus?"
- Any question that benefits from a segment-level view

## Analysis Steps
1. Group accounts by segment using employee_count thresholds
2. Calculate key metrics per segment: lead volume, MQL rate, SQL rate, pipeline, win rate, average deal size, ARR
3. Compare segment performance against targets or prior period
4. Identify over/under-performing segments
5. Cross-reference with product usage if relevant

## Output Format
- Segment comparison table
- Highlight the segment with highest growth potential
- Recommendation on where to invest
