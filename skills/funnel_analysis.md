# Skill: Funnel Analysis

## Description
Analyzes conversion rates across the lead-to-revenue funnel (Lead → MQL → SQL → Opportunity → Closed Won). Identifies where drop-off is highest and which segments or channels convert best.

## When to Use
- "How is the funnel performing?"
- "Where are we losing leads?"
- "What's our MQL-to-SQL conversion rate?"
- Any question about stage-to-stage conversion

## Analysis Steps
1. Count leads at each funnel stage for the requested time period
2. Calculate stage-to-stage conversion rates
3. Break down by segment, channel, or campaign if requested
4. Compare to prior period
5. Identify the stage with the largest drop-off
6. Recommend where to focus improvement efforts

## Output Format
- Summary conversion table (stage, count, conversion rate, change vs prior period)
- Highlight the biggest bottleneck
- Recommendation
